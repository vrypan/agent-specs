# CLI Spec Changelog

## 1.1.0 - 2026-05-01

- Added `OPT-COMPLETION` for optional bash, zsh, and fish shell completion.
- Removed shell completion generation from `EXCLUDE`.
- Added `FLAGS-VALUE-NAMES` to define recognized flag value placeholders.
- Specified that unrecognized flag value names are treated as generic `VALUE`.

## 1.0.0 - 2026-04-30

- Initial versioned CLI parser specification.
